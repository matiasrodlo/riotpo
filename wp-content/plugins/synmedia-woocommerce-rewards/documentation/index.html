Quick start

  1. Upload the plugin to your Wordpress and enable it.
  2. Go to WooCommerce->Settings->Rewards and enter the percentage of rewards to give.
  3. Click the checkbox "Enable the Rewards system" and Save the settings and voila!

Installation

IMPORTANT - ALWAYS BACKUP YOU SYSTEM BEFORE MAKING ANY UPGRADES OR CHANGES!

 

Rewards is a Wordpress Plugin which allows the WooCommerce plugin give rewards to customers.

Installation of this plugin is fairly simple. To install a WordPress Plugin automatically with the WordPress built-in installer, you need to manually upload this plugin to WordPress.

  1. Under your Wordpress admin view select "Plugins"
  2. From the submenus under the "Plugins" menu select "Add New"
  3. Select the "Upload" tab option from the links on the top of the "Install Plugins" page
  4. Press the "Browser" button to select the synmedia-woocommerce-rewards.zip file and press "Install Now" button
  5. Once installed go to your Plugins page again and activate the plugin "SYN Media Rewards for WooCommerce"

Updates

To update the plugin, follow these directions:

  1. Delete the old plugin folder
  2. Upload the new folder
  3. Make sure to check if the extension is activated

Usage

Follow these steps to setup your Rewards:

  1. In your WooCommerce Settings, select Rewards.
  2. Enter the title of your rewards, this title will be use again and again into your website.
  3. Enter the percentage of rewards to give to the customer.
  4. Select where to calculate the rewards, from the total, the subtotal or the total without shipping.
  5. Select where to show the Rewards amount that the customer will earn.
  6. Select where to show the button "Use my rewards for this order".
  7. To create the custom page "My rewards", click on the designated link in the Rewards Settings description.

Create "My rewards" page

To create the page "My rewards" for the customer, simply follow these instructions:

  1. Go to WooCommerce->Settings
  2. Click on the Rewards tab
  3. Click the link "To create the page "My rewards" into the site, click here"
  4. There should be a page called "My rewards" under the WooCommerce "My accounts" page.

Shortcode

Some shortcode have been made to simplify the process of integrating the Rewards with your template. A custom Wysiwyg button has been installed with this plugin to provide you an easy way of selecting them.

List of Shortcode:

[swr_cart_amount] : Will show the amount of rewards the customer will earn from his current cart

[swr_rewards_amount] : Will show how much rewards the customer has earned in their account.

[swr_view_rewards] : Will show a table of the rewards earn and spend for each order.

[swr_product_rewards] : Will show the current product rewards.

Functions

Some functions are available for advanced users:

Get the amount of rewards in the cart:

swr_get_cart_rewards();

 

Get the amount of rewards in the cart.

swr_get_current_rewards();

Get the current user rewards amount, can be pass an array of settings.

$current_rewards = swr_get_current_rewards(array(

'user_id' => 0, // ID of the user to return the rewards, 0 by default to return current user

'formatted' => false, // Only for Money, return a formatted currency (Ex.: 1,03 $) or not (1.03)

'convert_to_money' => false // Only for Points, return the Points in money value

));

 

Get the current product rewards that can be earned:

swr_show_product_reward();

 

Troubleshooting

I've clicked on the "To create the page "My rewards" into the site, click here" link but no page has been added.

First try to reclick on the link, if no page is created you can create it manually like any other pages. Just be sure to include the code [swr_view_rewards]
